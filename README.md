# Blog Preview Card

This is a simple Blog Preview Card project built using HTML and CSS.
This project is created as part of frontend practice to improve layout, styling, and box-shadow design.

## 📌 Project Overview

The Blog Preview Card displays:

* Blog image
* Category tag
* Published date
* Blog title
* Description
* Author profile

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* Box Shadow
* Border Radius

## 📁 Project Structure

```
BLOG PREVIEW CARD
│
├── index.html
├── style.css
├── illustration-article.svg
├── image-avatar.webp
└── README.md
```

## 🎨 Features

* Responsive card design
* Bottom-right shadow effect
* Clean layout
* Rounded corners
* Simple and modern UI

## 🚀 How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Open index.html in your browser
